# ShaktiDB Workspace

## Project Title

### ShaktiSOC

AI-Driven Security Operations Center using ShaktiDB

---

## Project Overview

ShaktiSOC is a cybersecurity monitoring platform designed to collect Linux system telemetry, store security events in ShaktiDB, and analyze system activities for threat detection and security analytics.

The project aims to explore database-driven security monitoring while evaluating ShaktiDB's capabilities for handling continuous event ingestion, storage, querying, and analysis.

The platform is planned to monitor:

* User Login Activity
* Process Execution
* Network Connections
* USB Device Activity
* File System Events

Collected events will be stored in ShaktiDB and analyzed to identify suspicious behavior and generate security alerts.

---

## Objectives

The goals of this project are:

* Install and explore ShaktiDB
* Develop a database-driven application
* Maintain source code using GitHub
* Document development progress
* Evaluate ShaktiDB performance and features
* Explore AI-assisted security analytics
