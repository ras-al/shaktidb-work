# ShaktiDB Workspace

## Project Title

### ShaktiSOC
AI-Driven Security Operations Center using ShaktiDB

---

## Project Overview

ShaktiSOC is an AI-powered cybersecurity monitoring platform designed to collect Linux system telemetry, store security events in ShaktiDB, and detect anomalous activities using machine learning techniques.

The platform continuously monitors:

- User login activity
- Process execution
- Network connections
- USB device activity
- File system events

Collected events are stored in ShaktiDB and analyzed by an AI-based threat detection engine to identify suspicious behavior and generate real-time alerts.
